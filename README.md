
A modern, AI-powered resume website with a live recruiter chatbot powered by Claude AI.

## Features
- Stunning dark-theme design with animated elements
- AI chatbot that answers recruiter questions about your background in real time
- Fully responsive (mobile + desktop)
- One-page with smooth scroll sections: Hero, Skills, Experience Timeline, Patents, Certifications, Contact
- Deployable to Vercel (free tier) in under 5 minutes

---

## Deployment Guide

### Step 1 — Get the Anthropic API Key
1. Go to https://console.anthropic.com and sign up / log in
2. Navigate to **API Keys** → **Create Key**
3. Copy the key (starts with `sk-ant-...`)
